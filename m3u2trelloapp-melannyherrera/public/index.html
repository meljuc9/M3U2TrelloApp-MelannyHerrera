Entregable
Avance 1 (Para el 16 de julio del 2022)

Debe contar con una estructura HTML base con columnas que hagan referencia a untablero tipo metodología Kanban (Tarea por Hacer, Tarea en Proceso, Tarea Finalizada), encabezado con el nombre del desarrollo, favicon de la aplicación web(opcional), formulario con los siguientes campos:
a.Título de la tarea.
b.Descripción de la tarea.
b.Responsable.
b.Fecha de entrega.
Brindar un estilo agradable usando Estilos CSS.

Avance 2 (Para el 23 de julio del 2022)

Hacer uso de las diferentes herramientas o librerías necesarias para añadir movimiento de la tarjetas para moverse entre cada columna a la que pertenezca, permitir el uso de JS para brindar la funcionalidad a la aplicación web capturando los datos en una fake API.

Avance 3 (Para el 31 de julio del 2022)

Tarea finalizada con todas sus funcionalidades y características necesarias para ser llamado un clon básico de trello donde adjuntamos todas nuestras tareas en nuestro tablero tipo metodología Kanban (Por Hacer, Haciendo, Hecho).
